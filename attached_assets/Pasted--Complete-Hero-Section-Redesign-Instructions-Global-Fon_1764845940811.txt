## Animation & Interaction Details

### Rotating Facts Animation
- Duration per fact: 4 seconds total (0.5s fade in + 3s display + 0.5s fade out)
- Transition: opacity ease-in-out
- Continuous loop through all three facts

### Typewriter Animation
- Typing speed: 80ms per character
- Backspace speed: 40ms per character
- Pause after complete word: 2s
- Cursor blink: 0.7s interval
- Cursor character: "|" or "▎"
- Cursor color: matches gradient (or solid #8B5CF6)
- Loop continuously through: ChatGPT → Perplexity → Gemini → Claude → Copilot

### Input Container Interactions
- Default state: border rgba(255, 255, 255, 0.1)
- Focus state: border rgba(59, 130, 246, 0.4), transition 0.3s ease
- Backdrop-filter: blur(20px) always active

### CTA Button Interactions
- Default state: gradient background, no shadow
- Hover state: 
  - Background gradient shifts slightly lighter
  - Box-shadow: inset 0 0 20px rgba(255, 255, 255, 0.1)
  - Transition: all 0.3s ease
- Active/Click state: slight scale down to 0.98

### Trust Badge
- Static (no hover effects needed)
- Subtle entrance animation on page load: fade in from bottom, 0.6s delay after page load

## Implementation Notes

- Use CSS custom properties for colors to maintain consistency
- Implement typewriter with JavaScript using async/await for clean timing control
- Use CSS Grid or Flexbox for centering, avoid absolute positioning where possible
- Ensure all animations use CSS transforms for performance (GPU acceleration)
- Add will-change: transform to animated elements
- Preload custom fonts to prevent FOUT (Flash of Unstyled Text)
- Add reduced-motion media query fallbacks for accessibility

**Design Philosophy:** Minimal, elegant, professional with subtle futuristic touches. Dark background allows gradient accents and light borders to create depth without visual clutter. Every element serves a purpose, nothing decorative.
